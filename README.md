# 番茄钟 - 时间管理工具

一个简单易用的番茄钟应用，帮助你提高工作效率。

## 功能特点

- ✅ **专注模式**：自定义时间的专注计时
- ✅ **短休息**：5分钟短暂休息
- ✅ **长休息**：15分钟长休息（每完成4个番茄后推荐使用）
- ✅ **自定义时间**：可自由设置专注、短休息、长休息的时间
- ✅ **可视化计时器**：清晰的倒计时显示
- ✅ **今日统计**：记录今日完成的番茄数
- ✅ **总完成数**：累计完成的番茄总数
- ✅ **深色模式**：一键切换深色/浅色主题
- ✅ **提示音**：计时结束时播放提示音

## 配色方案

- 🍅 **番茄色**：#ff8787（浅番茄色）
- 🌾 **米白色**：#fafafa
- ⬜ **白色**：#ffffff
- 🌙 **深色模式**：黑色背景 (#2d2d2d) + 白色文字

## 使用方法

### 启动应用

1. 确保已安装现代浏览器
2. 直接在浏览器中打开 `index.html` 文件

### 基本操作

1. **切换模式**：点击"专注"或"休息"按钮切换不同模式
2. **开始/暂停**：点击"开始"按钮开始计时，再次点击暂停
3. **重置**：点击"重置"按钮重新开始
4. **深色模式**：点击右下角的 🌙/☀️ 按钮切换主题
